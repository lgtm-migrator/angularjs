ES INFO

The info API provides general information about Build Model, Licencing Information and feature information of installed elasticsearch version .

Description

The information provided by this API includes

  • Build Information - including the build number and timestamp
  • License Information - basic information about the currently installed license
  • Features Information - JThe features that are currently enabled and available in the current license

Using INFO

A GET request that can be exposed through Web Browsers as well as from REST clients. Similarly to elasticsearch default context,the endpoint info will be used to use this API.
